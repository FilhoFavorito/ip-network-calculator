# IP-NETWORK-CALCULATOR


# Calculadora de Endereçamento IP

Aplicação que recebe um endereço IP no formato `IP/MÁSCARA` e retorna as principais informações da rede.



#### A aplicação deve receber como entrada duas informações:
* O endereço IP e máscara de rede, representado no formato IP/MR.
* Ex: "192.168.258.250/24".

#### A partir destas informações, deve apresentar:
- O endereço da rede
- O endereço de broadcast
- O primeiro endereço IP de hosts na rede
- O último endereço IP de hosts na rede
